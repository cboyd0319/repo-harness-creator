# Multi-Agent Orchestration

Use delegation only for independent research, review, or file ownership slices.

## Coordinator Duties

- Define objective, owned files, off-limits paths, acceptance criteria, and
  verification.
- Integrate and review results locally.
- Keep final correctness with the coordinator.

## Safety Rules

- Delegation does not relax platform, overwrite, redaction, or verification
  rules.
- No sub-agent should edit the same files as another active worker.
- Do not claim implementation exists until it is in this working tree and
  verified.
